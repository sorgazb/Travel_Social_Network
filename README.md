# Travel Social Network

![C++](https://img.shields.io/badge/C++-OOP%20%26%20LPI-00599c?style=for-the-badge&logo=cplusplus)&nbsp;![Cola](https://img.shields.io/badge/Estructura-Cola-orange?style=for-the-badge)&nbsp;![Ficheros](https://img.shields.io/badge/Lectura-Ficheros-green?style=for-the-badge)&nbsp;![POO](https://img.shields.io/badge/Programaci%C3%B3n-Orientada%20a%20Objetos-blueviolet?style=for-the-badge)

> **Travel Social Network** es un proyecto en C++ que simula una red social de viajes compartidos, implementando POO, LPI, estructura de Cola, y lectura de ficheros para gestionar conductores y viajes de forma dinámica.

---

## 📋 Descripción del Proyecto

**Travel Social Network** modela un sistema de transporte colaborativo donde conductores y viajeros interactúan mediante **Listas con Punteros Interiores (LPI)** y **estructuras de Cola** implementadas en C++.

Las funcionalidades principales incluyen:

- **Insertar** un nuevo conductor al sistema.
- **Valorar** a un conductor.
- **Insertar** un viaje nuevo.
- **Mostrar** la lista de conductores, viajes y toda la información conjunta.
- **Buscar** un conductor de la lista mediante su ID.
- **Eliminar** a un conductor de la lista.
- **Mostrar una cola** de todos los viajes con el mismo origen y destino.
- **Lectura de datos** desde ficheros para inicialización automática del sistema.

## ⚙️ Compilación y Ejecución

Clonar el repositorio:

```bash
git clone https://github.com/sorgazb/Travel_Social_Network.git
cd Travel_Social_Network/RSV
```

Compilar con g++:

```bash
g++ -o travel_social *.cpp
```

Ejecutar:

```bash
./travel_social
```
